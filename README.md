LISTA 02 - LAYOUTS COM FLEXBOX E GRID
Disciplina: Programação para Internet I

Aluno: Pedro Antonio

Objetivo: Aplicar conceitos avançados de CSS Flexbox e CSS Grid para replicar layouts estruturados e responsivos.

Exercício 1: Criei um menu de navegação horizontal usando Flexbox, utilizando justify-content para distribuir o logo e os links, e align-items para centralização vertical.

Exercício 2: Desenvolvi uma grade de 12 cards coloridos com Flexbox.

Exercício 3: Montei uma galeria de imagens estilo mosaico usando CSS Grid, controlando o tamanho das fotos com grid-column e grid-row para ocupar espaços diferentes.

Exercício 4: Estruturei um layout utilizando Grid Areas (grid-template-areas).

Exercício 5: Construí uma página completa combinando as duas técnicas: Grid e Flexbox.
